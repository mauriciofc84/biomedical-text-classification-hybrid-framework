## Description

This repository contains the official code and implementation of the hybrid framework proposed in our manuscript[cite: 1]. The architecture integrates explicit rule-based knowledge via automatically generated Regular Expressions (RegExes) with contextual text representations using traditional and Transformer-based models (Support Vector Machine, Naïve Bayes, BioBERT, and SetFit)[cite: 1]. To ensure transparent decision-making, an explainability protocol using SHAP and Contextual Importance and Utility (CIU) is provided[cite: 1].
